Desarrollador de Software &amp; Técnico Electrónico
# [mano robotica- lenguaje de senas]

## 📝 Descripción
*Mano robotica que ayuda a aprender el abecedario a jovenes con discapcaidad auditiva.*

---

## 🚀 Características Principales
- ✅ Innovacion en la educación.
- ✅ Diseño 3d.
- ✅ Algoritmo de control sencillo.
- ✅ Simulación completa verificada cada letra.

---

## 🛠️ Tecnologías y Herramientas
- **Hardware:** Arduino uno, Servos motores, controlador de servos pca9685, materiales de provecho.
- **Software/Firmware:** arduino, Framework Arduino, Python (para el Dashboard).
- **Simulación y Diseño:** Proteus 8.15, KiCad (Diseño de PCB).
